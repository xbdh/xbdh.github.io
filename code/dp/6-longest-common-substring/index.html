DP 6 longest common substring

发表于2021-03-11|更新于2021-03-11
|阅读量:

longest common substring

求最长公共子串的长度

brute-force

Time Complexity : O(3^(M+N) )

Space Complexity : O(N+M)

bottom-up

Time Complexity : O(M * N)

Space Complexity : O(N * M)

longest common subsequence

求最长公共子序列

brute-force

Time Complexity : O(2^(M+N) )

Space Complexity : O(N+M)

top-down

Time Complexity : O(M * N)

Space Complexity : O(N * M)

bottom-up

Time Complexity : O(M * N)

Space Complexity : O(N * M)

minimum deletions or insertions to transform a string into another

给定字符串S1,S2,通过删除插入字符使得S1变成S2，求最小的删除插入数

bottom-up

Time Complexity : O(M * N)

Space Complexity : O(N * M)

longest increasing subsequence

求最长递增子序列的长度

brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

top-down

Time Complexity : O(N^2 )

Space Complexity : O(N^2)

bottom-up

Time Complexity : O(N^2 )

Space Complexity : O(N)

max sum increasing subsequence

求 和最大的递增子序列

不同于题4

brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

top-down

三维表或哈希表

bottom-up

Time Complexity : O(N^2 )

Space Complexity : O(N)

shortest common super-sequence

给定字符串S1,S2,求最短的公共超级序列的长度，使得S1和S2都是其子序列

brute-force

Time Complexity : O(2^(M+N) )

Space Complexity : O(N+M)

top-down

Time Complexity : O(M * N)

Space Complexity : O(N * M)

bottom-up

Time Complexity : O(M * N)

Space Complexity : O(N * M)

minimum deletions to make a sequence sorted

给定数字序列，删除一些数，使得剩下的增序，求最小的删除数

同题4

brute-force

Time Complexity : O(N^2 )

Space Complexity : O(N)

longest repeating subsequence

求最长重复子序列，此子序列出现超过两次，索引位置不重复

brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

top-down

Time Complexity : O(N^2 )

Space Complexity : O(N^2)

bottom-up

Time Complexity : O(N^2 )

Space Complexity : O(N^2)

subsequence pattern matching

给定字符串S1，模式串S2，求S1中出现S2(作为序列)的次数

brute-force

Time Complexity : O(2^N ) ,N为S的长度

Space Complexity : O(N)

top-down

bottom-up

Time Complexity : O(M * N)

Space Complexity : O(N * M)

longest bitonic subsequence

求最长的双调子序列的长度

双调：先单调递增，再单调递减

brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

bottom-up

Time Complexity : O(N^2 )

Space Complexity : O(N)

longest alternating subsequence

求最长交替子序列的长度

模式：a> b <c ,a< b >c

brute-force

Time Complexity : O(3^(N+M) )

Space Complexity : O(N+M)

top-down

Time Complexity : O(N * M )

Space Complexity : O(N * M )

bottom-up
