unbounded knapsack

给定N个物品的价值和重量，一个容量为C的背包。每个物品不限次数且总容量不能超过C，求最大价值

Brute-force

Time Complexity : O(2^(N+C) )

Space Complexity : O(N + C)

Top-down

Time Complexity : O(N + C)

Space Complexity : O(N + C)

Bottom-up

Time Complexity : O(N + C)

Space Complexity : O(N + C)

selected item

rod cutting

给定长度为N的杆，切成小段，不同长度有不同的价格，求最大收益

同上题1

Bottom-up

Time Complexity : O(N + C)

Space Complexity : O(N + C)

selected item

coin change

无限量不同面值的零钱和要换的钱N，求有多少种换钱的方式

Brute-force

Time Complexity : O(2^(N+T) )

Space Complexity : O(N + T)

Top-down

Time Complexity : O(N * T)

Space Complexity : O(N * T)

Bottom-up

Time Complexity : O(N * T)

Space Complexity : O(N * T)

minimum coin change

无限量不同面值的硬币和要换的总钱N，求硬币总数最小值

Brute-force

Time Complexity : O(2^(N+T) )

Space Complexity : O(N + T)

Top-down

Time Complexity : O(N * T)

Space Complexity : O(N * T)

Bottom-up

Time Complexity : O(N * T)

Space Complexity : O(N * T)

maximum ribbon cut

相当于：无限量不同面值的硬币和要换的总钱N，求硬币总数最大值

Brute-force

Time Complexity : O(2^(N+T) )

Space Complexity : O(N + T)

Bottom-up

Time Complexity : O(N * T)

Space Complexity : O(N * T)
