DP 4 fibonacci numbers

发表于2021-03-11|更新于2021-03-11
|阅读量:

fibonacci numbers

斐波那契数列

Brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N)

Space Complexity : O(N)

staircase

n阶台阶，每次只允许走1步、2步或3步，求有多少种方式

Brute-force

Time Complexity : O(3^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N)

Space Complexity : O(N)

number factor

给定n,求有多少方式使得n等于{1,3,4}的和

Brute-force

Time Complexity : O(3^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N)

Space Complexity : O(N)

minimum jumps to reach the end

给定一组整数（>=0）,每个数代表可以往右跳的最大步数，求最小跳几次到达数组最后

元素为0时，不能跳

Brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N^2)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N^2)

Space Complexity : O(N)

minimum jumps with fee

给定N，代表n步台阶，N个元素的数组代表上某一台阶的花费，你每次可以走1,2,3步，求最小花费到达终点

Brute-force

Time Complexity : O(3^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N)

Space Complexity : O(N)

house thief

小偷偷房子，n元数组代表n个房子的财富，小偷不能偷连续2个房子，求能偷到的最大财富

Brute-force

Time Complexity : O(2^N )

Space Complexity : O(N)

Top-down

Time Complexity : O(N)

Space Complexity : O(N)

Bottom-up

Time Complexity : O(N)

Space Complexity : O(N)
